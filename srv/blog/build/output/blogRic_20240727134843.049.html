Notez que les URLs de cdnjs pour les fichiers CSS et JS peuvent changer en fonction des versions et des mises à jour. Veillez donc à vérifier les URLs avant de les utiliser dans votre projet. En ce qui concerne les logos de GitHub et Telegram, vous devez remplacer les URLs des images par les URLs de vos propres logos. Vous pouvez également personnaliser les tailles et les classes CSS des images pour les adapter à votre design.
